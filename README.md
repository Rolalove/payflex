# FlexPay

A premium, high-fidelity financial dashboard for managing **Split Bills** and **Escrow Transactions**. FlexPay offers a smooth, responsive experience for coordinating payments between multiple parties with secure, real-time verification.

## 🚀 Features

- **Split Bills**: Easily share costs with friends or colleagues with automated participant tracking.
- **Escrow Services**: Secure service agreements where funds are held safely until work is approved.
- **Interswitch Integration**: Full support for the Interswitch WebPay gateway for secure bank and card payments.
- **Real-time Notifications**: Invitations and status updates sent via email (Resend) and managed through Supabase.
- **Mobile-First Design**: A premium, responsive UI that works perfectly on desktop and mobile viewports.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Database & Auth**: [Supabase](https://supabase.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Payments**: [Interswitch WebPay](https://www.interswitchgroup.com/)
- **Emails**: [Resend](https://resend.com/)

## Getting Started

First, install the dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
